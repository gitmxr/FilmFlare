# 🎬 FilmFlare — Movie & Music Discovery Platform

FilmFlare is a dynamic web application built with **React.js** and **Tailwind CSS**, designed to let users explore trending **movies** and **music tracks** in one unified entertainment hub.  
Powered by **TMDb API** for movies and **YouTube Data API** for music, FilmFlare delivers real-time content discovery, search functionality, and media playback — all in a sleek, responsive interface.

---

## 🚀 Features

- 🎥 **Movie Discovery** — Browse trending, top-rated, Bollywood, and Hollywood movies in real-time.  
- 🎵 **Music Streaming** — Explore Indian, Pakistani, and English music categories with live YouTube integration.  
- 🔍 **Smart Search** — Search across movies and songs with instant results and smooth transitions.  
- 🎬 **Detailed Pages** — View trailers, details, and similar recommendations for both movies and music.  
- 🧭 **Pagination & Navigation** — Load more results easily with clean navigation and back functionality.  
- 📱 **Responsive UI** — Optimized for desktop, tablet, and mobile viewing.
